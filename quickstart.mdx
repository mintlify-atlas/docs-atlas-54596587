By the end of this guide you will have a working TypeScript program that simulates a full Generation I battle using `@pkmn/engine`, printing the outcome to the console.

## Prerequisites

- **Node.js 16 or later** — `node --version` to check
- **npm** — included with Node.js
- **Zig compiler (master branch)** — required to build the native addon at install time. The `postinstall` script will attempt to download one automatically if it cannot find a compatible version on your `PATH`.

<Warning>
  Only the Zig **master branch** is currently supported. The engine tracks upstream Zig closely, and the minimum stable release may not yet be sufficient. See [Installation](/installation) for details.
</Warning>

## Steps

<Steps>
  <Step title="Install the packages">
    `@pkmn/engine` ships a TypeScript driver that wraps the native engine code. It depends on `@pkmn/data` for Pokémon data and works with `@pkmn/dex` as the data source. Install all three:

    ```bash
    npm install @pkmn/engine @pkmn/dex @pkmn/data
    ```

    During installation, a `postinstall` script runs [`install-pkmn-engine`](https://github.com/pkmn/engine/blob/master/src/bin/install-pkmn-engine). This script locates (or downloads) a compatible Zig compiler and builds the native Node addon into `node_modules/@pkmn/engine/build/lib`.

    <Note>
      If you run npm with `--ignore-scripts`, the postinstall step is skipped. Run `npx install-pkmn-engine` manually afterward to build the addon.
    </Note>
  </Step>

  <Step title="Create your TypeScript file">
    Create a file called `battle.ts` and import the packages you need:

    ```ts
    import { Generations } from '@pkmn/data';
    import { Dex } from '@pkmn/dex';
    import { Battle, Choice, Result, initialize } from '@pkmn/engine';
    ```

    `initialize()` must be called before using `Battle` in browser environments to ensure the WASM module is ready. In Node.js it is optional but harmless to include.
  </Step>

  <Step title="Define your teams">
    The engine accepts teams as arrays of `Partial<PokemonSet>`. The example below uses the same teams as the official example in `examples/js/example.ts`, defined in Pokémon Showdown packed format and parsed with `@pkmn/sets`:

    ```ts
    import { Team } from '@pkmn/sets';

    const P1 = Team.unpack(
      'Fushigidane|Bulbasaur||-|SleepPowder,SwordsDance,RazorLeaf,BodySlam|||||||]' +
      'Hitokage|Charmander||-|FireBlast,FireSpin,Slash,Counter|||||||]' +
      'Zenigame|Squirtle||-|Surf,Blizzard,BodySlam,Rest|||||||]' +
      'Pikachuu|Pikachu||-|Thunderbolt,ThunderWave,Surf,SeismicToss|||||||]' +
      'Koratta|Rattata||-|SuperFang,BodySlam,Blizzard,Thunderbolt|||||||]' +
      'Poppo|Pidgey||-|DoubleEdge,QuickAttack,WingAttack,MirrorMove|||||||', Dex
    )!.team;

    const P2 = Team.unpack(
      'Kentarosu|Tauros||-|BodySlam,HyperBeam,Blizzard,Earthquake|||||||]' +
      'Rakkii|Chansey||-|Reflect,SeismicToss,SoftBoiled,ThunderWave|||||||]' +
      'Kabigon|Snorlax||-|BodySlam,Reflect,Rest,IceBeam|||||||]' +
      'Nasshii|Exeggutor||-|SleepPowder,Psychic,Explosion,DoubleEdge|||||||]' +
      'Sutaamii|Starmie||-|Recover,ThunderWave,Blizzard,Thunderbolt|||||||]' +
      'Fuudin|Alakazam||-|Psychic,SeismicToss,ThunderWave,Recover|||||||', Dex
    )!.team;
    ```

    Alternatively, you can construct team arrays directly from plain objects without `@pkmn/sets`.
  </Step>

  <Step title="Create the battle">
    Use `Battle.create()` with a `Generation` object and options describing both players and the RNG seed:

    ```ts
    const gens = new Generations(Dex);
    const gen = gens.get(1);

    const options = {
      p1: { name: 'Player A', team: P1 },
      p2: { name: 'Player B', team: P2 },
      seed: [1, 2, 3, 4],
      showdown: true,
      log: true,
    };

    const battle = Battle.create(gen, options);
    ```

    `showdown: true` runs the engine in Pokémon Showdown compatibility mode. `log: true` enables protocol message logging (the engine must also have been built with `-Dlog` for log data to be populated).
  </Step>

  <Step title="Run the battle loop">
    The engine is driven by calling `battle.update(c1, c2)` with both players' choices. After each update, call `battle.choices(player, result)` to get the valid choices for the next turn. The loop ends when `result.type` is truthy (i.e. non-zero):

    ```ts
    // A simple PRNG — replace with crypto.getRandomValues() in production
    class Random {
      seed: number;
      constructor(seed = 0x27d4eb2d) { this.seed = seed; }
      next(max: number) {
        let z = (this.seed += 0x6d2b79f5 | 0);
        z = Math.imul(z ^ (z >>> 15), z | 1);
        z = z ^ (z + Math.imul(z ^ (z >>> 7), z | 61));
        z = (z ^ (z >>> 14)) >>> 0;
        return Math.floor((z / 0x100000000) * max);
      }
    }

    const random = new Random();
    const choose = (choices: Choice[]) => choices[random.next(choices.length)];

    let result: Result, c1 = Choice.pass, c2 = Choice.pass;
    while (!(result = battle.update(c1, c2)).type) {
      c1 = choose(battle.choices('p1', result));
      c2 = choose(battle.choices('p2', result));
    }
    ```

    <Warning>
      Only pass choices returned by `battle.choices()` to `battle.update()`. Passing an invalid choice is **undefined behavior** and may corrupt the battle state or crash the engine.
    </Warning>
  </Step>

  <Step title="Read the result">
    The `Result` returned when the loop exits tells you who won. `result.type` is from the perspective of Player 1:

    ```ts
    const msg = {
      win:   'won by Player A',
      lose:  'won by Player B',
      tie:   'ended in a tie',
      error: 'encountered an error',
    }[result.type];

    console.log(`Battle ${msg} after ${battle.turn} turns`);
    ```
  </Step>
</Steps>

## Full example

The following is the complete working file, adapted directly from [`examples/js/example.ts`](https://github.com/pkmn/engine/blob/master/examples/js/example.ts):

```ts
import { Generations } from '@pkmn/data';
import { Dex } from '@pkmn/dex';
import { Battle, Choice, Log, Lookup, Result, initialize } from '@pkmn/engine';
import { Team } from '@pkmn/sets';

class Random {
  seed: number;
  constructor(seed = 0x27d4eb2d) { this.seed = seed; }
  next(max: number) {
    let z = (this.seed += 0x6d2b79f5 | 0);
    z = Math.imul(z ^ (z >>> 15), z | 1);
    z = z ^ (z + Math.imul(z ^ (z >>> 7), z | 61));
    z = (z ^ (z >>> 14)) >>> 0;
    return Math.floor((z / 0x100000000) * max);
  }
}

const P1 = Team.unpack(
  'Fushigidane|Bulbasaur||-|SleepPowder,SwordsDance,RazorLeaf,BodySlam|||||||]' +
  'Hitokage|Charmander||-|FireBlast,FireSpin,Slash,Counter|||||||]' +
  'Zenigame|Squirtle||-|Surf,Blizzard,BodySlam,Rest|||||||]' +
  'Pikachuu|Pikachu||-|Thunderbolt,ThunderWave,Surf,SeismicToss|||||||]' +
  'Koratta|Rattata||-|SuperFang,BodySlam,Blizzard,Thunderbolt|||||||]' +
  'Poppo|Pidgey||-|DoubleEdge,QuickAttack,WingAttack,MirrorMove|||||||', Dex
)!.team;

const P2 = Team.unpack(
  'Kentarosu|Tauros||-|BodySlam,HyperBeam,Blizzard,Earthquake|||||||]' +
  'Rakkii|Chansey||-|Reflect,SeismicToss,SoftBoiled,ThunderWave|||||||]' +
  'Kabigon|Snorlax||-|BodySlam,Reflect,Rest,IceBeam|||||||]' +
  'Nasshii|Exeggutor||-|SleepPowder,Psychic,Explosion,DoubleEdge|||||||]' +
  'Sutaamii|Starmie||-|Recover,ThunderWave,Blizzard,Thunderbolt|||||||]' +
  'Fuudin|Alakazam||-|Psychic,SeismicToss,ThunderWave,Recover|||||||', Dex
)!.team;

const gens = new Generations(Dex);
const gen = gens.get(1);
const options = {
  p1: { name: 'Player A', team: P1 },
  p2: { name: 'Player B', team: P2 },
  seed: [1, 2, 3, 4],
  showdown: true,
  log: true as true,
};

const play = () => {
  const battle = Battle.create(gen, options);
  const log = new Log(gen, Lookup.get(gen), options);
  const display = () => {
    for (const line of log.parse(battle.log!)) {
      console.log(line);
    }
  };

  const random = new Random();
  const choose = (choices: Choice[]) => choices[random.next(choices.length)];

  let result: Result, c1 = Choice.pass, c2 = Choice.pass;
  while (!(result = battle.update(c1, c2)).type) {
    display();
    c1 = choose(battle.choices('p1', result));
    c2 = choose(battle.choices('p2', result));
  }
  display();

  const msg = {
    win:   'won by Player A',
    lose:  'won by Player B',
    tie:   'ended in a tie',
    error: 'encountered an error',
  }[result.type];

  console.log(`Battle ${msg} after ${battle.turn} turns`);
};

// In Node.js, play() can be called directly.
// In the browser, initialize() must resolve first to load the WASM module.
if ('process' in globalThis) {
  play();
} else {
  initialize(options.showdown).then(play).catch(console.error);
}
```

Run it with:

```bash
npx ts-node battle.ts
```

Or compile first and run with Node:

```bash
npx tsc battle.ts --esModuleInterop --module commonjs
node battle.js
```
